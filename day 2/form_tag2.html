thẻ label là 1 thẻ nhãn thường được dùng trong thẻ form
     *** input có type="text" đây là 1 ô input để nhập liệu thông thường
    - các thuộc tính của thẻ input có type="text"
    1 - trong thẻ input luôn có thuộc tính type để chỉ ra kiểu của ô input
    2 - thuộc tính name="" : tên của ô input
    3 - thuộc tính value="" :chứa giá trị của ô input
    4 - placeholder="" : gợi ý cho người dùng phải gõ cái gì vào ô
    input và giá trị của placeholder chỉ hiện thị khi giá trị value của ô input bị rỗng
